# Created new project
```bash
npx nuxi init flash

yarn install

yarn dev
```


## Setup Firebase

Created new project on firebase

```bash
# yarn
yarn add firebase

yarn dev

```

## Setup Tailwind
```bash
yarn add --dev @nuxtjs/tailwindcss

yarn dev
```

Add this to `nuxt.config.ts`

```
 modules: [
    '@nuxtjs/tailwindcss'
]
````

## Setup Pinia
```bash
yarn add pinia @pinia/nuxt
```
